# email testing
[mailtrap](https://mailtrap.io) is a test mail server solution that allows testing email notifications without sending them to the real users of your application.

requires a few envs:
 ```
 MAIL_HOST=xxx
 MAIL_PORT=xxx
 MAIL_USER=xxx
 MAIL_PASS=xxx
 ```
